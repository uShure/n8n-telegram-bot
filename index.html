Как это работает:

  1. Telegram Trigger - Получает сообщения и callback от пользователей
  2. Router - Определяет тип запроса (команда /start или выбор роли)
  3. Google Sheets - Проверяет данные пользователя и лимиты
  4. Check Limits - Валидирует возможность генерации
  5. DeepSeek API - Генерирует текст объяснительной

Быстрая установка

1

Создайте Telegram бота

Используйте @BotFather для создания бота и получения токена

/newbot → Имя бота → Username → Сохраните токен
2

Настройте Google Sheets

Создайте таблицу с колонками: userId, chatId, role, quota, sub

Включите Google Sheets API → Создайте OAuth2 credentials
3

Получите DeepSeek API ключ

Зарегистрируйтесь на platform.deepseek.com и создайте API ключ

Dashboard → API Keys → Create New Key
4

Импортируйте workflow в n8n

Загрузите JSON файл и настройте credentials для всех сервисов

Import → Select File → Configure Credentials → Activate
